JavaScript Lesson 6

The focus of this lesson is while loops. We will use a while loop to print the first 10 square numbers when you click the button.

    This example uses a while loop to prompt the user to guess the program's name. It keeps running until the right guess is made (hint: it's Bob).

    Exercise 1

    Write the code to print the first 10 powers of 2 (starting with 0) using a while loop.

    Exercise 2

    Write a function to play a guessing game. The computer should think of a number from 1 to 100 and prompt the user to guess it. If the guess is wrong, the computer should tell the user if it was too high or too low. If the guess is right, the computer should tell the user how many guesses it took. You can use this code to generate the random number:

    var secret = Math.floor(Math.random() * 100) + 1
